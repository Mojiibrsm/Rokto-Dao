# **App Name**: Lifeline Hub

## Core Features:

- Donor Registration: Allow users to register as blood donors with their personal and contact information.
- Blood Drive Locator: Help users find nearby blood drives based on their location using reverse geocoding.
- Appointment Scheduling: Enable donors to schedule appointments for blood donation at specific locations and times.
- Donation History: Maintain a record of each donor's past donations, including dates and locations.
- Eligibility Checker: Ask a few basic health-related questions, and offer preliminary determination of eligibility, powered by a generative AI tool.
- Data Management: Use Google Sheets to store and manage donor information, blood drive details, and appointment schedules.

## Style Guidelines:

- Primary color: Strong, slightly desaturated red (#D92E2E) to symbolize blood and urgency.
- Background color: Very light, desaturated shade of red (#F2D4D4).
- Accent color: Light blue (#709FBA) as a complementary color for balance and trust.
- Body and headline font: 'PT Sans' for a clean and modern look.
- Use clear, recognizable icons for navigation and key features (e.g., a drop of blood, a calendar, a map marker).
- Maintain a clean, responsive layout that is easy to navigate on both desktop and mobile devices.
- Incorporate subtle animations for loading states and confirmations to provide feedback to the user.
